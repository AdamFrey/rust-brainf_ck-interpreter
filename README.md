# A Brainf*ck interpreter written in Rust

Inspired by a
[blog post](https://nickdesaulniers.github.io/blog/2015/05/25/interpreter-compiler-jit/)
by Nick Desaulniers

To run:
- `rustc -O bf.rs`
- add a bf program to the directory as `test.b`
- `./bf test.b`
